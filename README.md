# Hand_Gesture_Dino_Game
## Project Description

The **Hand Gesture Dino Game** is a computer vision–based machine learning project that allows users to control the Chrome Dino game using hand gestures instead of a keyboard. The system uses a webcam to capture real-time video of the user's hand and processes the video frames to detect and track hand movements.

Using hand tracking and gesture recognition techniques, the system identifies specific hand gestures and maps them to game controls such as jump and duck. When a gesture is detected, the program automatically sends the corresponding keyboard command to the game, enabling a touchless gaming experience.

This project demonstrates how machine learning and computer vision can be used to create interactive applications and improve human–computer interaction by allowing users to control systems through natural hand movements.

---

## Technologies Used

* **Python** – Programming language used for development
* **OpenCV** – For image processing and webcam video capture
* **MediaPipe** – For real-time hand detection and hand landmark tracking
* **NumPy** – For numerical operations and data processing
* **PyAutoGUI** – For simulating keyboard inputs to control the game
